# Interactive Quiz Platform

An internship-level quiz and assessment web application built using React and JavaScript.

## Features
- User name input
- Category-based quizzes
- Timer-based question handling
- Dynamic question rendering
- Real-time score calculation
- Quiz history using localStorage
- Responsive UI

## Tech Stack
- React
- JavaScript
- CSS
- Vite

## Run Locally

```bash
npm install
npm run dev
